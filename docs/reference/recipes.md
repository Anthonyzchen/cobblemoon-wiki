## Changed & Non-Craftable Recipes

**Changed recipes** — JEI shows the new recipe, but heads-up as some are intentionally pricier:  
- **PokéBike** — needs Ad Astra parts (Steel/Iron Plates, Wheels, Diamond), not cheap vanilla mats.  
- **Traveler's Backpack** — base + tier upgrades cost more (ender pearl + iron base; iron/gold/diamond blocks for the tiers).  
- **Reveal Glass** (the Mega Showdown form item, for Therian formes) — crafted with Diamonds + an Echo Shard.

**Not craftable anymore → loot only** (JEI will show NO recipe — this is intentional, not a bug):  
- **Ability Patch / Ability Capsule** — from **Raid Dens (Tiers 4–6)**, not a craft.  
- **Traveler's Backpack netherite tier + strongest attachments** (Void, Magnet, Crafting, Pickup, Feeding) — from **Epic crates / Hard chests**.
