## Form Changes

**Form changes (via Mega Showdown)**  
After you *catch* a Pokémon, swap forms with these items (catch path is in the dex posts).

- **Groudon → Primal Groudon**: Red Orb (Mega Showdown version)  
- **Kyogre → Primal Kyogre**: Blue Orb  
- **Dialga → Origin Forme**: Adamant Crystal  
- **Palkia → Origin Forme**: Lustrous Globe  
- **Giratina → Origin Forme**: Griseous Core  
- **Kyurem → Black / White Kyurem**: DNA Splicer + Zekrom or Reshiram in party  
- **Zacian → Crowned Sword**: Rusted Sword  
- **Zamazenta → Crowned Shield**: Rusted Shield  
- **Tornadus / Thundurus / Landorus / Enamorus → Therian Forme**: Reveal Glass (during a thunderstorm)  
- **Hoopa → Hoopa Unbound**: Prison Bottle (Mega Showdown's, not the M&L Hoopa Ring/Prison Bottle Key)  
- **Ogerpon → Cornerstone / Hearthflame / Wellspring Forme**: matching mask  
- **Calyrex → Ice Rider / Shadow Rider**: Reins of Unity + Glastrier or Spectrier in party  
- **Shaymin → Sky Forme**: Gracidea Flower  
- **Deoxys → Speed / Attack / Defense / Normal**: Deoxys Meteorite  
- **Necrozma → Dawn Wings**: N Lunarizer + Lunala in party  
- **Necrozma → Dusk Mane**: N Solarizer + Solgaleo in party  
- **Necrozma → Ultra Burst**: Dormant Crystal  
- **Zygarde → 10% / 50% / Complete**: Reassembly Unit + cells/cores (drop from Wishing Weald, Archaeological Sites, Observatory barrels)  
- **Terapagos → Terastal / Stellar**: Stellar Tera Shard (then terastallize in battle)  
- **Mewtwo → Mega Mewtwo X / Y**: Mewtwonite X or Y + Key Stone  
- **Rotom → Heat / Wash / Frost / Fan / Mow**: matching appliance block  
- **Oricorio → Baile / Pom-Pom / Pa'u / Sensu**: matching nectar (Red / Yellow / Pink / Purple)  
- **Gen 1-6 starters + many others → Mega Evolution**: matching mega stone + Key Stone

**Note:** some items exist twice — a **"… Key"** (M&L spawn key, summons the legendary) and the same-named **Mega Showdown** item (crafted, changes form post-catch). Example: the **Red Orb Key** spawns Groudon; the crafted **Red Orb** turns a caught Groudon Primal.
