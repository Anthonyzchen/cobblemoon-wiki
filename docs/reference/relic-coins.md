## Relic Coins (Currency)

**Relic Coins — the pack's currency**

**Earn them:**  
- **Loot balls & dungeon chests** — most tiers drop a few coins.  
- **Fishing** — ~15% chance of 1–2 coins per catch.  
- **Defeating trainers** — every defeat drops coins, scaling with the trainer's level band: 
	Lv 1–25 → 1–3 coins
	Lv 26–50 → 2–5 coins
	Lv 51–75 → 3–7 coins
	Lv 76–100 → 5–9  coins
- **FTB Daily Grind quests** (repeatable — the main source):
	110 coins/day (5 quests)
	200 coins/week (1 quest).  
- **FTB Adventure & Exploration milestones** — 6 relic coin sacks (~486 coins' worth) for completing exploration goals.

**Spend them at Merchants across various towns**
Stack coins into sacks (**9 Relic Coins → 1 Pouch**, **9 Pouches → 1 Sack**)

The most valuable are the crate keys from the **Crate Keymaster** villager in **Pallet Town**:  
- Common Key — **1 sack**  
- Uncommon Key — **2 sacks**  
- Rare Key — **6 sacks**  
- Epic Key — **15 sacks**  
- Legendary Key — **40 sacks**
