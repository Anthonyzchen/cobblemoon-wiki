## Riding

Rideable species show a 🐎 line with stats pulled directly from this pack's own data (Cobblemon species config + our datapack riding additions, plus Mega Showdown for its special-form mounts) — so everything listed is actually rideable here.  

**Types:** bird, horse, jet, dolphin, boat, submarine, hover, rocket.  

Riding improves through **Mount Mastery** as the Pokémon levels its mount stats.

**Stat legend** — each is a range, rolled per individual:  
Spd = Speed  
Acc = Acceleration  
Skl = Skill (handling / turning)  
Jmp = Jump  
Sta = Stamina
